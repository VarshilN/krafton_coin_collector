# krafton_coin_collector
A real-time multiplayer Coin Collector game  for the Krafton Associate Game Developer Assignment. Without using any networking engines or middleware, the project sets up a completely authoritative server, manual state synchronization, server-side collision detection, and a simulated 200ms network latency.
